## Философия

Я верю, что код — это не просто инструкции для машины. Это способ материализовать идеи, решать реальные проблемы и оставлять след. Каждый репозиторий здесь — не склад кода, а **глава в истории роста**.

**Мои принципы:**
- Без давления — системы поддержки, а не судьи
- Контекст важнее ярлыков
- Время как единственная валюта
- Осознанность вместо продуктивности

## Продукты

### AviaWallet
*Крипто-кошелёк с экосистемой*

[![App Store](https://img.shields.io/badge/App_Store-Available-black?style=flat-square&logo=apple)](https://apps.apple.com/ru/app/aviawallet/id6754718339)

Кроссплатформенное приложение: криптовалютный кошелёк, токен и экосистема. От идеи до App Store — путь ведущего разработчика.

### CREATMAN Life Hub
*Система управления жизнью как стратегической игрой*

Персональная веб-система для осознанной жизни. Не productivity tracker — интерфейс для ориентации в собственном пространстве.

`Python` `FastAPI` `Vanilla JS` `LLM Integration`
